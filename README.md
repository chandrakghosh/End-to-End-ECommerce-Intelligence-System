# End-to-End E-Commerce Intelligence System

## Project Overview

This project is an end-to-end E-Commerce Business Intelligence and Customer Intelligence Platform built using the Brazilian E-Commerce Public Dataset by Olist.

The project covers:

- End-to-End Data Pipeline
- Data Cleaning and Transformation
- Advanced EDA and Business Insights
- SQL Data Warehousing
- Machine Learning Models
- Power BI Dashboard
- Streamlit Web App
- Cloud Deployment

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- SQL / MySQL / PostgreSQL
- Scikit-learn
- Power BI
- Streamlit
- GitHub
